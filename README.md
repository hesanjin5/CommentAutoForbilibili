# 🎬 B站自动评论管理助手

> 一款专为B站用户设计的智能评论管理工具，支持评论抓取、本地素材库管理和定时自动发送功能。

[English](README_EN.md) | 中文

---

## ✨ 功能特性

### 📥 评论抓取
- 支持从任意B站视频评论区抓取评论
- 可选择 **最热评论** 或 **最新评论** 排序
- 支持分页抓取，每页最多19条评论
- 自动过滤空评论和无效数据

### 📦 本地素材库
- 默认容量 **200条** 评论，可自定义设置（10-1000条）
- 智能容量管理，自动淘汰旧评论（FIFO策略）
- 优先删除已使用过的评论
- 评论搜索功能，快速定位目标评论
- 评论使用状态跟踪（已使用/未使用）

### 🤖 定时发送
- **固定频次模式**：按设定间隔定时发送
- **伪随机模式**：在时间窗口内随机发送指定次数
- **两种选择方式**：
  - 按添加顺序依次发送
  - 随机抽取（不放回）
- 发送状态实时监控

### 🛡️ 风控防护
多重保护机制，模拟真人行为，最大程度降低封号风险：

- **发送频率控制**：随机延迟 ±20%~50%
- **时间段限制**：仅在活跃时段发送（默认9:00-23:00）
- **每日发送量限制**：默认50条/天
- **错误监控**：连续失败自动停止
- **内容多样性检查**：避免发送相似评论
- **浏览器环境模拟**：真实User-Agent和Headers

### 🎭 评论混淆
多种混淆方式，确保每次发送内容唯一：

- **零宽字符插入**：`\u200B`, `\u200C`, `\u200D`, `\uFEFF`
- **随机空格插入**：全角/半角空格随机分布
- **Emoji装饰**：15+预设emoji和特殊符号
- **同音字替换**：内置50+常用汉字同音字映射

## 🚀 快速开始

### 安装步骤

1. **安装油猴扩展**
   - [Tampermonkey (Chrome/Edge)](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - [Tampermonkey (Firefox)](https://addons.mozilla.org/firefox/addon/tampermonkey/)

2. **安装脚本**
   - 下载 [bilibili-comment-manager.user.js](bilibili-comment-manager.user.js)
   - 或在Tampermonkey中点击"创建新脚本"，将代码粘贴进去

3. **开始使用**
   - 访问任意B站页面
   - 点击右下角 📝 按钮打开控制面板

### 使用指南

#### 1. 评论抓取
1. 切换到 **评论抓取** 标签页
2. 输入目标视频链接（如：`https://www.bilibili.com/video/BV1xx411c7mD`）
3. 选择排序方式（最热/最新）
4. 设置抓取页数
5. 点击 **开始抓取**

#### 2. 素材库管理
1. 切换到 **素材库管理** 标签页
2. 查看当前缓存的评论列表
3. 使用搜索功能查找特定评论
4. 可进行以下操作：
   - 调整素材库容量
   - 重置评论使用状态
   - 清空素材库

#### 3. 定时发送
1. 切换到 **定时发送配置** 标签页
2. 选择发送模式（固定频次/伪随机）
3. 配置发送间隔或时间窗口
4. 设置评论选择方式
5. 配置混淆选项（建议全部启用）
6. 设置风控参数
7. 点击 **启动发送**

#### 4. 状态监控
1. 切换到 **运行状态** 标签页
2. 实时查看发送统计
3. 查看运行日志
4. 如有需要，导出日志进行排查

## ⚙️ 配置说明

### 发送模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 固定频次 | 按固定间隔发送 | 稳定、可预测的发送节奏 |
| 伪随机 | 在时间窗口内随机发送 | 更自然的行为模式 |

### 评论选择方式

| 方式 | 说明 | 特点 |
|------|------|------|
| 按添加顺序 | 按评论添加顺序发送 | 充分利用每条评论 |
| 随机抽取 | 随机选择未使用评论 | 更多样化，但可能浪费素材 |

### 混淆选项说明

| 选项 | 作用 | 影响 |
|------|------|------|
| 零宽字符 | 插入不可见字符 | 不影响阅读，但改变哈希值 |
| 随机空格 | 在文字间插入空格 | 可能轻微影响阅读 |
| Emoji装饰 | 添加emoji或符号 | 使评论更独特 |
| 同音字替换 | 替换部分汉字 | 可能影响语义 |

### 风控参数建议

| 参数 | 默认值 | 建议范围 | 说明 |
|------|--------|---------|------|
| 最小间隔 | 30秒 | 30-120秒 | 过短可能触发风控 |
| 每日限额 | 50条 | 30-100条 | 新账号建议保守 |
| 活跃时段 | 9-23时 | 8-24时 | 避免凌晨时段 |

## 🔧 技术栈

- **语言**：纯原生 JavaScript (ES6+)
- **存储**：localStorage
- **请求**：GM_xmlhttpRequest
- **框架**：无（纯DOM操作 + CSS Flexbox）
- **体积**：约 1700 行代码

### 为什么选择纯原生JS？

- ✅ 无需构建工具，即装即用
- ✅ 体积小，加载快
- ✅ 无外部依赖，兼容性好
- ✅ 用户可轻松修改源码

## ⚠️ 注意事项

### 安全使用建议

1. **新账号谨慎使用**
   - 建议先用小号测试
   - 初始每日限额设置较低（10-20条）

2. **合理设置参数**
   - 不要设置过短的发送间隔
   - 避开凌晨时段（0:00-6:00）
   - 控制每日发送量

3. **观察账号状态**
   - 关注是否有异常提示
   - 遇到错误码-509时立即暂停
   - 连续失败5次后自动停止

### 常见错误码

| 错误码 | 含义 | 处理建议 |
|--------|------|---------|
| 0 | 成功 | - |
| -101 | 未登录 | 检查Cookie |
| -509 | 请求频繁 | 增大发送间隔 |
| 12002 | 评论关闭 | 更换目标视频 |
| 12016 | 包含敏感词 | 修改评论内容 |
